# IPL Data Analysis Pipeline (Pandas & NumPy)

## Overview
This project focuses on analyzing IPL cricket data using both Pandas and NumPy. The project was designed to simulate a real-world data pipeline workflow including data ingestion, cleaning, transformation, analysis, reporting, and export.

The analysis was performed using the IPL datasets:
- `deliveries.csv`
- `matches.csv`

---

## What I Have Done
- Loaded and processed large IPL datasets
- Cleaned and validated match and ball-by-ball data
- Merged multiple datasets for unified analysis
- Performed data analysis using both Pandas and NumPy
- Calculated player and team statistics such as runs, strike rates, economy rates, boundaries, dot balls, and over-wise scoring
- Analyzed powerplay and death-over performances
- Generated venue-wise, city-wise, and season-wise insights
- Compared match outcomes and team performances
- Created structured reports and exported results into CSV and Excel files

---

## Technologies Used
- Python
- Pandas
- NumPy

---

## Project Outcome
This project helped in understanding:
- End-to-end data pipeline design
- Data cleaning and preprocessing
- Vectorized analysis using NumPy
- Aggregation and analysis using Pandas
- Working with real-world structured datasets
- Report generation and data export
